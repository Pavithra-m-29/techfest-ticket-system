🚀 Setup Guide

Firebase + Netlify — Step by step | TechFest Ticket System
📁
உங்களுக்கு கிடைத்த Files
index.html
🌐 Public Booking Page
யாரும் access பண்ணலாம்
admin.html
🔐 Admin Dashboard
Password protected
இந்த 2 files-ஐ ஒரே folder-ல வையுங்க → Netlify-ல upload பண்ணுங்க
1
🔥 Firebase Project Create பண்ணுங்க

1. console.firebase.google.com → Google account-ல login பண்ணுங்க

2. "Add project" click → Name: techfest2025 → Continue → Create

3. Left sidebar → "Realtime Database" → Create database → Start in test mode → Enable

4. Project Settings (⚙️ icon) → "Your apps" → Web app () icon → App nickname: ticketsys → Register

5. Firebase config copy பண்ணுங்க — இப்படி இருக்கும்:

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "techfest2025.firebaseapp.com", databaseURL: "https://techfest2025-default-rtdb.firebaseio.com", projectId: "techfest2025", storageBucket: "techfest2025.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc123" };
2
📝 Config இரண்டு Files-லயும் Paste பண்ணுங்க

index.html மற்றும் admin.html — இரண்டிலும் இந்த line தேடுங்க:

// ✅ YOUR FIREBASE CONFIG — Replace with yours from Firebase Console const firebaseConfig = { apiKey: "YOUR_API_KEY", ← இதை replace பண்ணுங்க ... };

YOUR_API_KEY, YOUR_PROJECT எல்லாத்தையும் உங்கள் actual config-ல உள்ள values-ஆல replace பண்ணுங்க.

✅ இரண்டு files-லயும் same config paste பண்ணணும்!
3
🌐 Netlify-ல Deploy பண்ணுங்க

1. netlify.com → Sign up (Free) → Login

2. Dashboard → "Add new site" → "Deploy manually"

3. உங்கள் folder-ஐ (index.html + admin.html இருக்கிற folder) drag & drop பண்ணுங்க

4. Deploy ஆகும் → உங்களுக்கு link கிடைக்கும்:

🔗 Public: https://techfest2025.netlify.app ← Users-க்கு share பண்ணுங்க 🔐 Admin: https://techfest2025.netlify.app/admin.html ← நீங்க மட்டும் use பண்ணுங்க

Custom domain வேணும்னா: Domain settings → Add custom domain (optional)

4
✅ Test பண்ணுங்க
  • Public link open பண்ணி ticket book பண்ணுங்க
  • Admin page open → admin / admin123 → Login
  • Dashboard-ல ticket real-time-ல தெரியணும் 🎉
  • Verify section-ல ticket ID enter பண்ணி check பண்ணுங்க
✅ யாரும் WhatsApp-ல link share பண்ணி ticket book பண்ணலாம்!
🔐
Admin Password மாத்தணும்னா

admin.html-ல இந்த lines தேடுங்க:

const ADMIN_USER = 'admin'; ← உங்கள் username const ADMIN_PASS = 'admin123'; ← உங்கள் password மாத்துங்க

உங்களுக்கு பிடிச்ச password வையுங்க, பிறகு Netlify-ல re-upload பண்ணுங்க.

🎉 Done! உங்கள் ticket system இப்போ worldwide accessible!
