# 🎟️ Secure Ticket Verification System

## 📌 Description
This project is a secure ticket booking and verification system using QR code and face verification to prevent fraud.

## 🚀 Features
- QR-based ticket generation
- Face verification (duplicate prevention)
- SHA-256 hashing for security
- Email ticket delivery
- Entry gate validation system

## 🛠️ Tech Stack
- Python Flask
- SQLite
- OpenCV
- SHA-256
- QR Code
- HTML, CSS, JavaScript
